# Sources

Raw source files go here. These are **immutable** — the LLM reads them but never modifies them.

## How to add sources

1. Copy or save the file into this directory (markdown, text, PDF, images)
2. Run Claude Code in the wiki directory: `cd ~/wiki && claude`
3. Tell Claude: "ingest sources/your-file-name.md"

## Supported formats

- `.md` — Markdown articles, notes (preferred)
- `.txt` — Plain text
- `.pdf` — PDF documents
- Images — PNG, JPG (Claude will read them visually)

## Tips

- Use [Obsidian Web Clipper](https://obsidian.md/clipper) to save web articles as markdown
- Keep original filenames descriptive
- One source per file
