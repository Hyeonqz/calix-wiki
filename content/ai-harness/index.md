# AI Harness

AI 코딩 에이전트와 플러그인 생태계에 대한 지식 정리.

Claude Code, Codex 같은 코딩 에이전트 위에 올라가는 harness(플러그인, 에이전트 프레임워크)를 다룬다.

## Plugins

코딩 에이전트의 워크플로우를 확장하는 플러그인 시스템.

- [Oh My ClaudeCode (OMC)](/ai-harness/plugins/omc) — 멀티 에이전트 오케스트레이션 레이어
- [OMX](/ai-harness/plugins/omx) — Claude Code 확장 플러그인
- [Superpowers](/ai-harness/plugins/superpowers) — 스킬 기반 워크플로우 자동화

## Agents

자율적으로 작업을 수행하는 AI 에이전트 프레임워크.

- [Hermes Agent](/ai-harness/agents/hermes) — 자기 개선형 AI 에이전트
- [Ouroboros](/ai-harness/agents/ouroboros) — 진화적 개발 루프 에이전트
